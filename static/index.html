Project Details

Built around your trained best.pt model

01

Purpose

This project turns a YOLO training result into a usable browser workflow so a user can test the model without touching the command line.

02

Image workflow

Users upload a single image, choose confidence and image size settings, then receive an annotated result with a detection table.

03

Video workflow

Users upload a video and the backend runs frame-by-frame inference, returning a processed video with bounding boxes.

About The Project

From training artifact to practical AI demo

The site is designed for a YOLO model trained in Ultralytics format. The backend loads the model once, reuses it for image and video inference, and keeps generated outputs in a local folder for easy review during development.
